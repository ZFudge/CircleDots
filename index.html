Controls
  • P : Pause / Resume
  • R : Resets both Dots and Rectangles
  • V : Toggle drawing lines On / Off (Off by Default)
  • 1-9 : Select background image

  • Dot Specific Controls
    • I : Rotate all dots counter-clockwise
    • O : Randomize rotation on all dots
    • P : Rotate all dots clockwise
    • G : Toggle Gravity Type(Circular, Cross); (Circular by Default)
    • T : Remove all dots on screen (does not affect rectangles)
    • Gravity Specific
      • E : Toggle inward / outward rotation
      • F : Set even rotation - (overrides inward / outwards rotation)
  • Rectangle Specific Controls
    • W : Toggle rectangles On / Off
    • Q : Toggle oscillation on screen / passing through screen
    • A : Toggle vertical / horizontal direction
    • J : Set direction left / up
    • K : Randomize direction
    • L : Set direction right / down (hold to keep shapes at edge of range)
